# Sales Dashboard

## Description

This project was developed using Microsoft Excel to analyze sales data and generate business insights. The dashboard calculates product-wise sales, revenue, and overall performance using Excel formulas.

## Features

* Sales Revenue Calculation
* Product-wise Sales Analysis
* Total Revenue Summary
* Data Visualization and Reporting
* Business Performance Tracking

## Products Included

* Mobile
* Laptop
* Earbuds

## Formulas Used

* Multiplication Formula (Quantity × Price)
* SUM()
* SUMIF()

## Tools Used

* Microsoft Excel 2016

## Outcome

The dashboard provides clear insights into product sales performance and revenue generation, helping users understand business trends effectively.
